Detailed Description

template<typename Templ_Type, unsigned int Templ_Size>
class Vector< Templ_Type, Templ_Size >

Szablon opisujacy wektor w przestrzeni. Za jego pomoca opisywane sa wierzcholki dowolnej figury jako wektory o poczatku w poczatku ukladu wspolrzednych. Wektory skladaja sie z tablicy typu double, w ktorej przechowywane sa kolejne skladowe.

Dla klasy zdefiniowano podstawowe przeciazenia operatorow pozwalajace na korzystanie z danych, ich modyfikacje oraz podstawowa arytmetyke wektorowa.
